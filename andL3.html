You must do a bitwise AND.

Start with the first bit of each of the two numbers, and AND them. This gives the first bit of the answer. Then AND the second bits together. This gives the second bit of the answer. Repeat this for each of the eight bits.

Using the truth table

For example, to use the truth table to find 0 AND 0 look for the cell which is in the column headed by 0 and is also in the row beginning with 0.

This is shown in the truth table below.

0 1
0 0 0
1 0 1

So 0 AND 0 = 0
